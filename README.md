## dotfiles

<pre>
put dotfiles in:
/rep/dotfiles

sudo mkdir /rep
sudo chown -R m:m /rep
cd /rep
git clone https://github.com/mort1skoda/dotfiles.git
</pre>


<pre>
create symlinks from ~ directory to
/rep/dotfiles
Thenit will reflect any adjustments / updates
to your dotfiles.

remember to:
cd /rep/dotfiles
git pull

git add

git commit

git push

So that updates are copied to github.
</pre>
